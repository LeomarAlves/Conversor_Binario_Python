# Conversor de Bases Numéricas

Este é um projeto simples em Python que permite a conversão de números entre diferentes bases numéricas (Decimal, Binário e Hexadecimal). O projeto conta com uma interface gráfica moderna desenvolvida com a biblioteca `customtkinter`.

## 🚀 Funcionalidades

- Conversão de **Decimal** para **Binário** e **Hexadecimal**.
- Conversão de **Binário** para **Decimal** e **Hexadecimal**.
- Conversão de **Hexadecimal** para **Decimal** e **Binário**.
- Interface gráfica intuitiva e responsiva.
- Suporte a temas claro/escuro (baseado no sistema).

## 🛠️ Tecnologias Utilizadas

- **Python 3**: Linguagem principal.
- **CustomTkinter**: Para a interface gráfica.
- **Tkinter**: Base para a interface.

## 📁 Estrutura do Projeto

- `main.py`: Contém a lógica de conversão matemática para as bases.
- `Interface.py`: Contém a implementação da interface gráfica e gerencia a interação do usuário.

## 🔧 Como Executar

1. **Clone o repositório** (ou baixe os arquivos):
   ```bash
   git clone <url-do-repositorio>
   cd PythonProject
   ```

2. **Instale as dependências**:
   Certifique-se de ter o Python instalado. Recomenda-se o uso de um ambiente virtual.
   ```bash
   pip install customtkinter
   ```

3. **Execute o aplicativo**:
   ```bash
   python Interface.py
   ```

## 📝 Como Usar

1. Selecione a **base de entrada** no menu suspenso superior.
2. Digite o número que deseja converter no campo de texto.
3. Selecione a **base de saída** no segundo menu suspenso.
4. Clique no botão **Converter** ou pressione **Enter**.
5. O resultado será exibido na parte inferior da tela.
